 📊 Projeto de Business Intelligence - Análise de Vendas
 
 📌 Sobre o projeto

Este projeto tem como objetivo transformar dados brutos em informações estratégicas, utilizando técnicas de modelagem de dados, SQL e Power BI.

O desenvolvimento foi realizado desde o tratamento dos dados até a construção de dashboards interativos para análise de indicadores de negócio.

---

## 🛠️ Tecnologias utilizadas

* SQL Server
* Power BI
* DAX

---

## 🔎 Etapas do projeto

### 🧩 1. Modelagem e tratamento de dados (SQL)

* Identificação de redundâncias em tabelas
* Criação de JOINs para consolidação dos dados
* Desenvolvimento de views otimizadas:

  * vwProduto
  * vwLoja
  * vwVendas
  * vw_taxa_cambio

---

### 📅 2. Criação da tabela calendário (DAX)

* Construção de tabela de datas
* Análise temporal dos dados
* Comparação de métricas ao longo do tempo

---

### 📊 3. Desenvolvimento do dashboard

* KPIs principais:

  * Receita
  * Lucro
  * Margem
  * Volume de vendas
* Evolução da receita ao longo do tempo
* Top produtos por desempenho
* Análise por país e categoria

---

## 📈 Resultados

* Modelo de dados mais eficiente e organizado
* Melhor performance nas análises
* Visualização clara para apoio à tomada de decisão
